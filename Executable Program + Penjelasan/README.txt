1. Software lain yang dibutuhkan untuk menjalankan executable:
   Java JDK / Java Runtime Environment (JRE) versi 8
   Eclipse IDE
   Apache Ant 

2. Cara instalasi program
   - Install Java Development Kit (JDK) 8
   - Install Eclipse IDE
   - Install Apache Ant
   - Install Virtenio SDK beserta library dari PreonVM 
    

3. Cara menggunakan executable program setelah diinstall
   - Pastikan seluruh node sensor telah aktif dan menggunakan modul autostart 
   - Menjalankan executable program dengan menjalankan java -jar Controller.jar pada command prompt.
   - Setelah aplikasi Controller terbuka, maka pilih menu yang sesuai 
     (1) Cek Node Aktif 
     (2) Sinkronisasi Waktu
     (3) Kirimkan Waktu 
     (4) Lakukan Sensing!
     (0) Keluar
   Catatan: 
   - Pastikan COM Port yang digunakan sesuai 
   - Pastikan konfigurasi jaringan (PAN ID) sesuai untuk seluruh node
